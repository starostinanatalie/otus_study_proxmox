# otus_study_proxmox
Ход выполнения домашнего задания

Терраформ был установлен ранее в ходе выполнения первого домашнего задания

1. Создала файл main.tf с указанием провайдера Proxmox
2. Выполнила инициализацию terraform с помощью команды terraform init
3. Написала в main.tf код для создания ВМ, для указания пароля для подключения к серверу использовала переменные в variables.tf, сами переменные из окружения
4. Проверила синтаксис с помощью terraform validate (исправила отмеченные ошибки)
5. Проверила правильность плана с помощью terraform plan
6. Создала ВМ с помощью команды terraform apply
